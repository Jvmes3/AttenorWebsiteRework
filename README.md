# Attenor Website Rework

## Run Locally

This is a static website. No install step is required.

1. Open a terminal.
2. Go to the project folder:

```bash
cd AttenorWebsiteRework

```bash
python3 -m http.server 8000

Open the site in your browser:
http://localhost:8000
To stop the server, press Ctrl + C in the terminal.
