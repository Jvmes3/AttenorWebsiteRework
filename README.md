# Attenor Website Rework

## Run Locally

This is a static website. No install step is required.

1. Open a terminal.
2. Go to the project folder:

-Bash <br>
cd AttenorWebsiteRework

-bash <br>
python3 -m http.server 8000


Open the site in your browser:
http://localhost:8000
To stop the server,<br> press Ctrl + C in the terminal.


significance + changes: <br>
This rework transforms Attenor’s website from a basic informational site into a professional client-conversion platform for a strategic foresight consultancy. The updated version features clearer service positioning, a modern responsive design, a downloadable resource hub, and a structured client funnel that guides visitors from email capture to discovery call booking.
